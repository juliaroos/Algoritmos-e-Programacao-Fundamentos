## ▶️ How to Run

To compile and run any `.c` file locally, make sure you have a C compiler installed (such as **GCC**).

### Compile

```bash
gcc file_name.c -o program
```

### Run

**Windows**

```bash
program.exe
```

**Linux / macOS**

```bash
./program
```
